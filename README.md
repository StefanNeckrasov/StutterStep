# English:
StutterStep – Make your connection stumble. This Fabric mod for 1.21.1 adds realistic high‑ping and lag simulation with per‑packet granularity.
Features:
• Master switch
• Configurable delay (0–3000 ms) + jitter
• Selective lag for attacks, movement, interactions, block breaking
• Admin command /stutterstep
• HUD icon
• Full config sync (Cloth Config)
• Function "Lag Progression"

# Русский:
StutterStep – Заставьте ваше соединение спотыкаться. Этот мод для Fabric 1.21.1 добавляет реалистичную симуляцию высокого пинга и лагов с точностью до типа пакета.
Возможности:
• Главный переключатель
• Настраиваемая задержка (0–3000 мс) + джиттер
• Выборочные лаги для атак, движения, взаимодействия, разрушения блоков
• Админская команда /stutterstep
• Иконка на HUD
• Полная синхронизация настроек (Cloth Config)
• Функции «Прогрессия лагов»
